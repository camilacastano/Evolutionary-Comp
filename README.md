# Evolutionary Computation

This repository presents full exercises related to the *Evolutionary Computation* course from my PhD classes. More than the content (I'm aware that I cannot share fully), I'll be uploading my assignments, with a Python script and a PDF explaining the development and problem we need to solve. The idea is to practice as much as I can and see whether it's possible to implement certain algorithmic structures, or their natural-evolution-related counterparts, such as Selection, Survival of the Fittest, Elitism, or Crossover, in my PhD thesis.

*On this main README, I will divide each task and assignment into parts, following the course division based on our trimester model.*
